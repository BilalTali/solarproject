## PHASE 2 — SECURITY AUDIT (CRITICAL — FIX EVERYTHING)

### 2A. Debug Code Removal

```bash
# Backend — find and REMOVE every instance:
grep -rn "dd("         app/ resources/ --include="*.php"
grep -rn "dump("       app/ --include="*.php"
grep -rn "var_dump("   app/ --include="*.php"
grep -rn "print_r("    app/ --include="*.php"
grep -rn "phpinfo()"   . --include="*.php"
grep -rn "ray("        app/ --include="*.php"
grep -rn "die("        app/ --include="*.php"
grep -rn "exit("       app/Http/ --include="*.php"
grep -rn "Log::debug(" app/ --include="*.php"

# Frontend — find and REMOVE every instance:
grep -rn "console\.log("    frontend/src/ --include="*.tsx" --include="*.ts"
grep -rn "console\.warn("   frontend/src/ --include="*.tsx"
grep -rn "debugger"         frontend/src/ --include="*.tsx"
grep -rn "alert("           frontend/src/ --include="*.tsx"
```

Fix rules:
- `dd()`, `dump()`, `var_dump()`, `print_r()` → DELETE the line
- `console.log()` → DELETE every occurrence
- `debugger` → DELETE immediately
- `alert()` → replace with toast notification
- `console.error()` in catch blocks → wrap: `if (import.meta.env.DEV) console.error(...)`

### 2B. SQL Injection Audit

```bash
# Find any raw query interpolation — every result must be fixed:
grep -rn "DB::select(\"" app/ --include="*.php"
grep -rn 'DB::select("'  app/ --include="*.php"
grep -rn "DB::statement(" app/ --include="*.php"
grep -rn "whereRaw(\$"   app/ --include="*.php"
grep -rn "orderByRaw(\$" app/ --include="*.php"
grep -rn "selectRaw(\$"  app/ --include="*.php"
grep -rn "havingRaw(\$"  app/ --include="*.php"
```

For every result:
- If using `$request->input()` directly inside raw SQL → **THIS IS INJECTION** → fix immediately
- Use `DB::select('...WHERE id = ?', [$id])` or Eloquent methods
- Never concatenate user input into SQL strings

### 2C. Mass Assignment Audit

```bash
# Find dangerous mass assignment patterns:
grep -rn "'\$guarded'" app/Models/ --include="*.php"
grep -rn '"role"' app/Models/User.php
grep -rn '"is_admin"' app/Models/ --include="*.php"
```

Rules:
- `$guarded = []` → **FORBIDDEN** in any model. Use `$fillable` explicitly instead.
- `role`, `is_admin`, `status` MUST NOT be in any model's `$fillable`
- Verify all controllers use `$request->validated()` not `$request->all()`

```bash
# Find controllers using request->all():
grep -rn "->create(\$request->all())"  app/Http/Controllers/ --include="*.php"
grep -rn "->fill(\$request->all())"    app/Http/Controllers/ --include="*.php"
grep -rn "->update(\$request->all())"  app/Http/Controllers/ --include="*.php"
```

Fix every instance: replace `$request->all()` with `$request->validated()`.

### 2D. File Upload Security

Find all file upload handlers:
```bash
grep -rn "->file(" app/ --include="*.php"
grep -rn "->store(" app/ --include="*.php"
grep -rn "UploadedFile" app/ --include="*.php"
```

For each upload:
- Verify MIME type is checked server-side (not just extension)
- Verify file size limit is enforced
- Verify files are stored in `storage/` not `public/`
- Verify file is served through signed URLs, not direct paths

Correct pattern:
```php
$request->validate([
    'document' => 'required|file|mimes:jpg,jpeg,png,pdf|max:5120',
]);
$path = $request->file('document')->store('documents', 'local');
```
